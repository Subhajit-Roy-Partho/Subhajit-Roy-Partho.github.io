### VScode shortcuts for Linux:
- **Fold innermost** - Ctrl + Shift + [
- **Unfold innermost** - Ctrl + Shift + ]
- **Fold All** - Ctrl + K, Ctrl + 0
- **Unfold All** - Ctrl + K, Ctrl + J
